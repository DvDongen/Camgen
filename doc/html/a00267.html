Detailed Description

template<class value_t, std::size_t N_in, std::size_t N_out>
class Camgen::helicity_generator< value_t, N_in, N_out, false >

Helicity generator base class specialization for discrete helicities.

A derived class should provide an implementation of the generate() and (optionally) evaluate_weight() member functions and have constructor which takes two arrays of numbers or a process_tree list iterator as arguments.
